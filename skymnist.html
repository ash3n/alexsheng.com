SkyMNIST: The World’s First Self-Reprogramming AI

Sorry if my writing is bad. This isn’t a formal write up, just a quick recap of some research.

The Adventure Begins

Last month was winter break for most colleges in the US. I had a month on my hands with no schoolwork to grind through or parties to be afraid of missing. That’s 30*24 = 720 free hours, or more relevantly 30*16 = 480 waking hours. So, I decided to pull the trigger on the one thing that would keep me interested this winter: I invented self-reprogramming artificial intelligence.

Hence, I humbly present SkyMNIST, the first AI in human history that learns to rewrite its own source code.

Breakthroughs in my Parents’ Garage

I’ve found that when you want to build new technologies that don’t yet exist, everything that can go wrong along the way will go wrong along the way. You want to build it up piece by piece and iterate – one small victory at a time to keep space for course corrections and to preserve your own sanity. For self-reprogramming AI, I decided to start with the basics. For AI, the basics would be MNIST digit recognition, which is kind of like the “Hello World” of machine learning.

So here’s the general outline. This AI is composed of two primary components: a classification module and a self-awareness module. The classification module learns the digit recognition task, and the self-awareness module learns to write computer code and reprograms the source code of the classification module to improve its capabilities ad infinitum.

The implementation of the classification module is straightforward. It’s just a neural network trained for image classification on the MNIST dataset. We can throw it into Keras and train it on 1000 images of MNIST and see how quickly it learns to classify. We let the form of this neural network be decided by the source code written by the self-awareness module. For simplicity, I’m only using fully-connected feedforward layers for now.

For the self-awareness module, we start with a transformer neural network that can write computer code. We could train an AI from scratch to learn computer programming, but we don’t necessarily have to go through that hassle. There are already pretrained computer programming AI models that we can download and retrain for our own purposes. My first thought was OpenAI’s Codex model, but OpenAI isn’t the most open about letting outside researchers use their models. I instead went with Salesforce’s CodeT5, which was openly available and lightweight enough to train on my mom’s laptop.

To train the self-awareness module, I created a dataset of neural network source code examples in Python, then randomly generated commands like “insert a new layer with 512 nodes and ReLU activation” and procedurally applied them. I trained the self-awareness module to learn this procedure, so that if I give it a Python source code and a command, it will apply that command to the source code and give me a new refined source code.

Judgement Day

Now, everything is more or less ready. Let’s put our self-reprogramming AI to work. I’ll keep it simple and just tie everything together with a basic iterative search algorithm. I start with a nonexistent classification module, and let the self-awareness module build it from the ground up. I give the self-awareness module an empty string as the initial source code, and randomly generate refinement commands to try. The self-awareness module spits out a bunch of refined source codes, and the algorithm compiles and tests them out one by one. The source code is used to build the classification module, which is then trained on 1000 images to see how well it learns image classification. The one that gets the best average accuracy while training over those 1000 datapoints is kept as the new source code for the classification module. We take this new classification module, and feed its source code back into the self-awareness module to rinse and repeat. Over many iterations, we end up with a very effective classifier.

We now have a functioning prototype, and there’s definitely a lot of things to be added. Next up, I’ll be replacing the random sampling and procedural parts of our process with decisions from generative models to get a smarter iterative refinement algorithm.

Thoughts

This AI isn’t fully self-referential since it’s built out of two separate modules with one module rewriting the other. It isn’t likely to achieve Skynet-or-Ultron-esque exponential development into an invincible superintelligence, so humanity should be safe for the time being. I’m only publishing the details of this preliminary study right now, but I’m currently testing a generalized version of this technology that is self-referential and theoretically capable of exponential development. I will be releasing details of my research in stages as time goes on, with an increasing focus on safety and alignment with human values.

Just like that, we’ve built the world’s first self-reprogramming AI. It’s primitive and rough around the edges, but this is the start of a technological breakthrough that will shape our future. Cheers!

~ Alex
